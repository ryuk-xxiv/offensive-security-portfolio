# Sanitized Report Samples

This directory contains templates and sanitized examples for penetration testing and vulnerability assessment reporting.

## Recommended Report Structure

1. Cover Page
2. Executive Summary
3. Scope and Methodology
4. Key Strengths
5. Key Weaknesses
6. Risk Rating Methodology
7. Findings Summary
8. Detailed Findings
9. Remediation Roadmap
10. Appendix

## Finding Template

### Finding Title

**Severity:** Critical / High / Medium / Low / Informational  
**Affected Asset(s):** Sanitized  
**Category:** Active Directory / Network / Web / Host / Cloud  

#### Description
Explain the issue in business and technical terms.

#### Impact
Describe what an attacker could realistically achieve.

#### Evidence
Include sanitized screenshots, commands, or observations.

#### Remediation
Provide clear prioritized remediation steps.

#### References
Include vendor or framework references when appropriate.
