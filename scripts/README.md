# Security Automation Scripts

This directory is reserved for safe automation scripts used for reporting, parsing, validation, and workflow improvement.

## Planned Scripts

- Vulnerability report parser
- Finding template generator
- Asset inventory cleaner
- CSV deduplication utility
- Port scan result summarizer
- Evidence folder organizer
- PowerShell local security configuration checker

## Safety Principles

Scripts published here should not include credentials, client data, destructive actions, exploit code, or live-target information.
